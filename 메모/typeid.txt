typeid

1. 자료형이나 변수 또는 식을 입력받아 const type_info& 형식의 객체를 반환해 주는 연산자

2. 자료형,클래스, 함수, 객체, enum,사용자 정의 클래스, 멤버 함수, 람다 함수까지 받을수있음

3. type_info 객체를 받아와서 .name()함수를 사용시 해당 자료형이 무엇인지 문자열로 반환

4. auto 같은 키워드로 인해 해당 자료형을 유추하기 힘든 변수의 자료형을 확인하고자 할 때 이 키워드를 사용하면 디버깅 하기 쉬움


사용예시

#include <iostream>
using namespace std;
void main(){
cout<< typeid(int).raw_name() << endl;
cout<< typeid(type_info).raw_name() << endl;
cout<< typeid(main).raw_name() << endl;
cout<< typeid(cout).raw_name() << endl;
}
