char* -> int : char* cstr = "20170711";
	      int num = atoi(cstr);
사용함수 : atoi() -> char 내부에 숫자만 넘기는데 예를들어 123abc456이 있으면 123만 넘기고 abc뒤의 456은 버려진다, 그리고 숫자가 하나도 없다면 0이 넘어감(오류는 나지않음)
