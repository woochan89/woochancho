특징

1. 객체를 생성하는 작업이 어렵거나 특정한 절차를 따라야 할 경우 이를 대행하는 함수를 두는 설계방식
2. 객체를 생성하되 직접 객체 생성자를 호출해서 객체를 생성하는게 아니라 대행함수를 통해 간접적으로 객체를 생성하는 방식 ex) gamestart()->(맵생성,캐릭터생성) 이런식으로 main에서 직접 맵과 캐릭터를 생성하는게 아니라 gamestart라는 함수를 통해 순차적으로 함수를 실행하는것
3. 객체 생성을 대행해주는 함수를 팩토리 메소드(Factory Method)라고 한다
4. 부모 클래스에서 자식 클래스가 어떤 객체를 생성할지 결정하도록 하는 패턴 -> 부모 클래스를 통해 객체 생성 함수(가상함수)를 호출하되 실제로는 부모의 가상함수를 거쳐 자식의 가상함수에서 실제로 생성함
