전역변수

특징

1. 함수 외부에 선언된다
2. 프로그램의 시작과 동시에 메모리 공간에 할당되어 종료시 까지 존재한다
3. 별도의 값으로 초기화 하지 않으면 0으로 초기화 된다
4. 프로그램 전체 영역 어디서든 접근이 가능하다


static 변수

특징

1. 선언된 함수 내에서만 접근이 가능하다
2. 딱 1회 초기화 되고 프로그램 종료시 까지 메모리 공간에 존재한다
3. 초기화 하지 않아도 0으로 초기화 된다


차이점

1. 함수를 빠져나가는 순간 값이 사라지는데 그 값이 사라지지 않고 유지하고 싶을때 static사용
2. 어디서든 접근 가능한 변수를 만들때 전역 변수를 사용한다(다만 오류발생시 찾기가 힘드므로 사용자제)




Static 멤버변수/함수

특징

1. 클래스 내에서의 전역변수 개념
2. 하나의 정적 멤버 변수는 같은 클래스 내 모든 객체가 공용으로 사용 가능->전역변수가 여러 함수에서 사용되는것과 비슷
3. 정적 멤버 변수도 하나의 객체에서 바꾼 값은 유지되며, 같은 정적 멤버 변수를 쓰는 다른 객체에 영향을 줌
4. 정적 멤버 변수는 클래스 내 모든 객체가 공용으로 사용할 수 있다
5. 정적 멤버 변수의 저장 원리는 전역변수와 동일
6. 정적 멤버 변수는 해당 클래스명으로 접근해야하며, private 정적 멤버 변수는 정적 멤버 함수로 접근함

※ 정적 멤버변수는 클래스 내에 [static 자료형 변수이름;]형식으로 정의
※ 정적 멤버변수는 클래스 밖에서 초기화 해야한다
※ 정적 함수에선 정적 변수만을 사용해야함
