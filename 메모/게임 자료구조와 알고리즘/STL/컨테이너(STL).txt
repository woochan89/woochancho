특징

1. 같은 타입을 저장,관리할 목적으로 만들어진 클래스
2. 시퀸스 컨테이너, 연관 컨테이너 두 가지가 있다
3. 데이터 저장 방식에 따라 배열 기반, 노드 기반으로 나뉘다

시퀸스 컨테이너 : 컨테이너 원소가 자신만의 삽입위치를 가지는 컨테이너(선형적)->vector,deque,list
연관 컨테이너 : 저장 원소가 삽입 순서와 다르게 특정 기준에 의해 자동 정렬되는 컨테이너(비선형,트리)-> set,multiset, map, multimap

배열 기반 컨테이너 : 데이터 여러개가 하나의 메모리 단위에 저장 -> vector, deque
노드 기반 컨테이너 : 데이터 하나를 하나의 메모리 단위에 저장 -> list, set, multiset, map. multimap
